=== 使用说明 ===
* 把所有文件（.tex, .m, .svg, .png,）放在 "littleshi.cn\\PhysWikiOnline\\" 目录下, 其中 PhysWiki.tex 文件用于生成目录。
* 搜索 "break point here", 然后在这些行设置 break point. 然后用 debug 模式运行。
* 在 PhysWiki.tex 中不存在的词条也会被转成 html, 但不会在 online 目录中出现。

=== 自己的 Note ===
* 如果要 debug 某一个文件的编译， 搜索 "one file debug"， 输入文件名设置 break point 即可。
